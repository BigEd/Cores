The bitfile is for a Nexys4ddr board available from Digilent.

Use at your own risk!

Switches 1 and 7 should be on. Switch 7 enables/disables interrupts. Switch 1 displays / hides sprites.

On the test system:

    A USB keyboard is connected to the USB Host port.

    Port JB is connected to a Pmod SDCard.

    Port JA (the lower set of pins) is connected to a Pmod RTCC

The system should be able to run without these devices connected.

The VGA is setup for a 1366x768 monitor display. Your monitor must be able to support this resolution in order to display correctly.

There are two indicators on-screen that flash. The first one indicates that timer IRQ's are happening. The second one indicates that CPU#1 is running in addition to CPU#0.

It may be necessary to press the reset switch several times in order to get the core to reset properly. There is something flaky about the reset process.

Boot Strapping

The LoadFromSerial port routine can be invoked by typing:

    J 12568 (this has changed)

in the monitor.

It will then load a 16k program into the system's memory at address $24000. Note address $20000 to $20007 cannot be used for code at the moment as that's also the variable for the flashing display. So any file will need eight bytes padding.
