RTF65002 System

I will not be working on the RTF65002 so much anymore; it's replaced by the RTF65003.

Check out the RTF65003. It's much the same as the '002 but it has better support for byte and 16 bit addressabilty.

Use at your own risk.

The bit file is for an Atlys FPGA board from Digilent.

The video output is setup for a 1366x768 HDMI display, your display device must be able to support this mode.

 

You may have to press the reset button to get the core started.

Press ALT-tab to switch between running tasks.

Pressing 'B' (uppercase B) at the prompt will start TinyBasic as a task. You can then ALT-tab over to it.

Pressing 'b' (lowercase b) at the prompt will start EhBASIC as a task. You can then ALT-tab over to it.

Pressing '?' at the prompt will display a list of commands.

Pressing CTRL-ALT-DEL will reset the system.

The system may start with an extra '.' after the '$' prompt at startup. Just delete it.

 

On the Atlys board:

Switch #7 globally disables/enables interrupts. The switch should be set to on, or the keyboard won't work.

Swithc #6 globally disables/enables the bitmap display device.

Switches #0 and #1 switch the video display between bitmapped and text mode. For text mode both switches should be on.

A Pmod SD can be attached to Port JB pins 1-6.

A PmodTMP can be attached to Port JB pins 7 to 12.
