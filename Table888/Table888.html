Hi, Welcome to the Table888 processor site.

Table888 is a work-in-progress.

Table888 is a 64 bit cpu with 256 general purpose registers. The Table888 book contains more details on the processor.

 

Features:

- 256 general purpose registers

- fixed length 40 bit instructions

- load/store architecture , memory mapped I/O

- two addressing modes - register indirect with displacement and indexed

 

Status:

- the processor is running a bootrom in an FPGA
